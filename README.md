## 📂 项目列表

### 🛒 项目一：电商用户行为分析与RFM用户分层

**技术栈：** Python (pandas/numpy/matplotlib/seaborn/scikit-learn) · SQL · Apriori算法

**项目内容：**
- 对电商平台20+用户行为数据（浏览/加购/购买）进行全流程分析
- 数据清洗与EDA探索性分析：品类销售分布、渠道用户画像、转化率分析
- **RFM模型**用户分层：将用户划分为高价值/发展/保持/挽留四类
- **Apriori关联规则挖掘**：发现品类间交叉销售机会
- **转化漏斗分析**：识别用户流失关键环节

**文件：**
- `ecommerce_analysis.ipynb` — 完整Jupyter Notebook（可直接运行）
- `ecommerce_data.csv` — 模拟电商用户行为数据集
- `eda_analysis.png` — EDA可视化
- `rfm_segmentation.png` — RFM分层结果可视化
- `conversion_funnel.png` — 转化漏斗图

### 📈 项目二：上市公司财务分析与杜邦模型

**技术栈：** Python (pandas/numpy/matplotlib/scikit-learn) · 财务建模 · 线性回归

**项目内容：**
- 选取美的、格力、海尔三家家电龙头近3年财务数据
- **杜邦分析模型**：ROE拆解为净利率×资产周转率×权益乘数
- **财务趋势分析**：营收、利润、毛利率、ROE、资产负债率趋势对比
- **现金流预测**：线性回归预测2025年经营现金流
- **行业对标雷达图**：多维度可视化三家公司财务竞争力

**文件：**
- `financial_analysis.ipynb` — 完整Jupyter Notebook（可直接运行）
- `financial_data.csv` — 三家上市公司近3年财务数据集
- `dupont_analysis.png` — 杜邦分解可视化
- `financial_trends.png` — 财务趋势图
- `cashflow_prediction.png` — 现金流预测图
- `radar_comparison.png` — 行业对标雷达图
