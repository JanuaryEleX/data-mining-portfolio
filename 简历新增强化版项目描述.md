## 项目经历推荐写法

### 用户流失预测模型 | Python · scikit-learn · 随机森林
*个人数据挖掘项目 | GitHub作品集*
- 基于用户行为数据构建流失预测模型，特征包含活跃天数、最近登录间隔、历史订单数、投诉次数、近30日支付次数等
- 完成数据清洗、特征工程、训练集/测试集划分，训练逻辑回归与随机森林模型
- 使用 Accuracy、Precision、Recall、AUC、混淆矩阵评估模型表现，并绘制 ROC 曲线
- 输出 Top30 高风险流失用户名单，为优惠券召回、客服回访等运营策略提供依据

### SQL用户运营分析 | SQL · SQLite · pandas
*个人业务分析项目 | GitHub作品集*
- 设计 users、products、orders、events 四张业务表，模拟互联网电商数据仓库结构
- 使用 SQL 计算月度 GMV、订单数、客单价、渠道付费转化率、复购率和7日留存率
- 构建浏览-加购-支付转化漏斗，识别用户流失关键环节
- 输出 TOP 品类 GMV 贡献分析，为渠道投放和品类运营提供数据支持

### 电商用户行为分析与RFM分层 | Python · RFM · Apriori
*个人数据分析项目 | GitHub作品集*
- 基于电商用户行为日志完成数据清洗、EDA探索性分析和可视化展示
- 构建 RFM 用户价值分层模型，将用户划分为高价值、发展、保持、挽留四类
- 使用 Apriori 关联规则挖掘品类组合关系，输出交叉销售建议
- 分析用户浏览-加购-支付转化漏斗，提出优惠券召回、品类推荐等运营建议
